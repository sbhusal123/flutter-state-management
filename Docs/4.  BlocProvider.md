# Bloc Provider:

- Bloc Provider is used to pass the state to the Widget Tree (or child widget) from the parent widget.

- Use it when the state of an application needs to be maintained at the higher level hierarcy in widget tree.



## 1. Wrap the parent widget with BlocProvider


```dart

import 'package:flutter/material.dart';
import 'package:flutter_bloc/flutter_bloc.dart';

import 'package:src/bloc/counter_bloc.dart';
import 'package:src/bloc/counter_event.dart';
import 'package:src/bloc/counter_state.dart';


void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',
      theme: ThemeData(colorScheme: .fromSeed(seedColor: Colors.deepPurple)),

      home: BlocProvider( // instiantiate the BlocProvider Here,
        // lazy: false -> false by default
        create: (context) => CounterBloc(),
        child: MyHomePage(title: "Flutter Demo Home Page"),
      )
    );
  }
}
```

Here, we've wrapped the widget with ``BlocProvider`` which accepts two parameters:


```dart
      home: BlocProvider( // instiantiate the BlocProvider Here,
        create: (context) => CounterBloc(),
        child: MyHomePage(title: "Flutter Demo Home Page"),
        lazy: false // false by default
      )
```

Here, ``BlocProvider`` disposes the ``CounterBloc`` automatically, and **is only created** whenever the child widget 
on a widget tree looks up with, ``context.read<CounterBloc>().``. 

**To instiate it without considering the lookup behaviour, use `lazy=true`**

**If the provider is already instiantiated**,  we can pass it directly as below:

```dart
      home: BlocProvider.value(
        value: context.read<CounterBloc>(), // already instiaited by some parent widget
        child: MyHomePage(title: "Flutter Demo Home Page"),
      )
```


## 2. Usage Within the Child Widget:

In the child component we dont need to instiantiate the ``Bloc``. 


**BlocBuilder:** 

```dart
            BlocBuilder<CounterBloc, CounterState>(
              // buildWhen: (previousState, currentState){
              //     print("Previous Value: ${previousState.count}");
              //     print("Current State: ${currentState.count}");
              //     if (currentState.count % 2 == 0) {
              //       return true;
              //     }
              //     return false;
              //   // return true;
              // },
              builder: (context, state) {
                return Text(
                  '${state.count}',
                  style: Theme.of(context).textTheme.headlineMedium,
                );
              },
            ),
```

**To emmit an event:**

```dart
          FloatingActionButton(
            onPressed: (){
              context.read<CounterBloc>().add(CounterDecreamentEvent()); // emmit an event
            },
            .....
```

**Full Code Sample**


```dart

class MyHomePage extends StatefulWidget {
  const MyHomePage({super.key, required this.title});

  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        backgroundColor: Theme.of(context).colorScheme.inversePrimary,
        title: Text(widget.title),
      ),
      body: Center(
        child: Column(
          ...,
          children: [
            const Text('You have pushed the button this many times:'),
            BlocBuilder<CounterBloc, CounterState>(
              // buildWhen: (previousState, currentState){
              //     print("Previous Value: ${previousState.count}");
              //     print("Current State: ${currentState.count}");
              //     if (currentState.count % 2 == 0) {
              //       return true;
              //     }
              //     return false;
              //   // return true;
              // },
              builder: (context, state) {
                return Text(
                  '${state.count}',
                  style: Theme.of(context).textTheme.headlineMedium,
                );
              },
            ),
          ],
        ),
      ),
      floatingActionButton: Row(
        ...,
        children: [
          FloatingActionButton(
            onPressed: (){
              context.read<CounterBloc>().add(CounterIncreamentEvent()); // emmit an event
            },
            .....
          ),
          FloatingActionButton(
            onPressed: (){
              context.read<CounterBloc>().add(CounterDecreamentEvent()); // emmit an event
            },
            .....
          ),          
        ],
      )
    );
  }
}
```
