# BlocListner

Listens to state changes in a bloc and executes certain logic in response to state changes such as 
showing `SnackBar`, `Dialog`.

Note that the BlocListner doesnt rebuilds thw whole UI unlike `BlocBuilder`. It runs exactly once for a response
to state change.

```dart
BlocListener<BlocA, BlocAState>(
  listenWhen: (previous, current) {
    // return true/false to determine whether or not
    // to invoke listener with state
  },
  listener: (context, state) {
    // do stuff here based on BlocA's state
    // navigate or display a SnackBar.
    // nullable
  },
  child: Container(), // child container: Should not be nullable
)
```

Note that:
- `listenWhen` returns a `boolean` value.
- `listener` does have any return type i.e. void.
- It expects a non nullable: `child`, instead of a nullable value use `Container` if needed.

```dart
    BlocListener<CounterBloc, CounterState>(
      listenWhen: (previous, current) {
        return true;
      },
      listener: (context, state) {
        if(state.count == 3){
          final snackbar = SnackBar(
            content: Text('Counter value is ${state.count}')
          );
          ScaffoldMessenger.of(context).showSnackBar(snackbar);
          // Note that this doesnt returns anything, return type => void
        }
      },
      child: SizedBox(
        width: 0,
        height: 0,
      ),
    )
```
