# State Event And BLoC:


## State:

Define a state class with a constructor and a property.

Example: 

```dart
class CounterState {
  final int count;

  const CounterState({required this.count});
}
```

## Event:

Define a event class.

Example: 

```dart
class CounterIncreamentEvent {}
```

## Bloc:

Define a bloc class with a constructor and a handler for each event.

Note that, it must extend from ``Bloc<<Event>, <State>>``

Constructor must initialize with initial state.


```dart
class CounterBloc extends Bloc<CounterIncreamentEvent, CounterState> {

  // constructor 
  CounterBloc() : super(const CounterState(count: 0)) {

    // event handler
    on<CounterIncreamentEvent>(
      (event, emit) => emit(CounterState(count: state.count+1))
    );
  }
}
```

## Usage With Flutter: 

```dart
import 'package:flutter/material.dart';
import 'package:flutter_bloc/flutter_bloc.dart';

import 'package:src/bloc/counter_bloc.dart';
import 'package:src/bloc/counter_event.dart';
import 'package:src/bloc/counter_state.dart';


void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      ...
      ...
      home: const MyHomePage(title: 'Flutter Demo Home Page'),
    );
  }
}

class MyHomePage extends StatefulWidget {
  const MyHomePage({super.key, required this.title});

  final String title;

  @override
  State<MyHomePage> createState() => _MyHomePageState();
}

class _MyHomePageState extends State<MyHomePage> {

  // initialize bloc class instance
  final counterBloc = CounterBloc();

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      ...
      body: Center(
        child: Column(
          ...
          children: [
            ...,
            BlocBuilder<CounterBloc,CounterState>( // it listens for state changes
              bloc: counterBloc,
              builder: (context, state) { // this function will call when state changes and rebuil a UI.
                return Text(
                  '${state.count}',
                  style: Theme.of(context).textTheme.headlineMedium,
                );
              },
            ),

          ],
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: (){
          counterBloc.add(CounterIncreamentEvent()); // 
        },
        ...
        ...
      ),
    );
  }
}
```
