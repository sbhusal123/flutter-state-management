# Setup and Codebase Structure:



**Packages Being used:**

- [Dio for performing Http Requests](https://pub.dev/packages/dio)
- [Flutter Bloc for State Management](https://pub.dev/packages/flutter_bloc)


```sh
flutter pub add dio flutter_bloc
```

[ReqRes](https://reqres.in/api/users?page=1) Users API: `https://reqres.in/api/users?page=1` fetchd with `Dio`.

[Quicktype](https://quicktype.io/) to model the JSON response into `UserModel`.



## 1. Folders / Code Structure:

```dart
├── lib
│   ├── constants
│   │   └── config.dart // api configs, api_key, ...
│   ├── data
│   │   ├── model
│   │   │   └── user_model.dart // JSON to Domain model
│   │   ├── provider
│   │   │   └── user_provider.dart // API Calls
│   │   └── repository
│   │       └── user_repository.dart // Repository and Interface
│   └── main.dart
```

- **Model:** This consists of object model mapped from API Response. Generated using [Quicktype](https://quicktype.io/).

- **Provider:** Consists of api calls i.e. data providers.

- **Repository:** Abstract repository and implementation

```dart
import 'dart:convert';

UserModel userModelFromJson(String str) => UserModel.fromJson(json.decode(str));

String userModelToJson(UserModel data) => json.encode(data.toJson());

class UserModel {
    int page;
    int perPage;
    int total;
    int totalPages;
    List<Datum> data;
    Support support;
    Meta meta;

    UserModel({
        required this.page,
        required this.perPage,
        required this.total,
        required this.totalPages,
        required this.data,
        required this.support,
        required this.meta,
    });

    factory UserModel.fromJson(Map<String, dynamic> json) => UserModel(
        page: json["page"],
        perPage: json["per_page"],
        total: json["total"],
        totalPages: json["total_pages"],
        data: List<Datum>.from(json["data"].map((x) => Datum.fromJson(x))),
        support: Support.fromJson(json["support"]),
        meta: Meta.fromJson(json["_meta"]),
    );

    Map<String, dynamic> toJson() => {
        "page": page,
        "per_page": perPage,
        "total": total,
        "total_pages": totalPages,
        "data": List<dynamic>.from(data.map((x) => x.toJson())),
        "support": support.toJson(),
        "_meta": meta.toJson(),
    };
}

class Datum {
    int id;
    String email;
    String firstName;
    String lastName;
    String avatar;

    Datum({
        required this.id,
        required this.email,
        required this.firstName,
        required this.lastName,
        required this.avatar,

    factory Datum.fromJson(Map<String, dynamic> json) => Datum(
        id: json["id"],
        email: json["email"],
        firstName: json["first_name"],
        lastName: json["last_name"],
        avatar: json["avatar"],

    Map<String, dynamic> toJson() => {
        "id": id,
        "email": email,
        "first_name": firstName,
        "last_name": lastName,
        "avatar": avatar,
    };
}
```

> **provider/user_provider.dart**

```dart
import 'package:app/data/model/user_model.dart';
import 'package:dio/dio.dart';

import 'package:app/constants/config.dart';

class UserProvider {
  final Dio _dio = Dio(
    BaseOptions(
      baseUrl: 'https://reqres.in/api/', 
      headers: {
        'x-api-key': apiKey
      }
    ),
  );

  Future<UserModel> getUsers() async {
    try {
      final response = await _dio.get('users/?page=1');
      return userModelFromJson(response.toString());
    } catch (e) {
      return Future.error(e.toString());
    }
  }
}
```

**repository/user_repository.dart**

```dart
import 'package:app/data/model/user_model.dart';
import 'package:app/data/provider/user_provider.dart';


// contract for repository implementation
abstract class IUserRepository {
  Future<UserModel> getUsers();
}


// repository implementation
class UserRepository implements IUserRepository {

  final UserProvider userProvider;

  UserRepository(this.userProvider);

  @override
  Future<UserModel> getUsers() {
    return userProvider.getUsers();
  }
}
```
